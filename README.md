# Dragnet Haul

Detection rules and IOC feeds generated by [Dragnet](https://github.com/dragnet-dev/dragnet).
Updated every 6 hours. Browse at [dragnet.dev](https://dragnet.dev).

---

## Contribute an incident

Add a YAML file to the relevant module's `incidents/` directory and open a PR.
See [schema documentation](https://github.com/dragnet-dev/dragnet) for the format and required fields.

Trusted sources auto-merge within 60 minutes. Others are reviewed manually.

---

## How it works

The sync workflow runs every 6 hours:

1. Downloads the latest `dragnet` binary from [dragnet-dev/dragnet releases](https://github.com/dragnet-dev/dragnet/releases)
2. Validates all incident YAML files against the schema
3. Pulls from configured intelligence sources (OSV, OSSF, NVD, CISA, PolySwarm, etc.)
4. Runs cross-domain enrichment (links actor profiles to incidents)
5. Generates detection rules and STIX 2.1 exports for all backends
6. Commits the updated data and opens draft PRs for newly detected incidents

The engine source lives in [dragnet-dev/dragnet](https://github.com/dragnet-dev/dragnet).
